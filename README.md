# Simple WebPage – HTML Assignment

## Description
This project is a basic HTML assignment created to practice core HTML concepts.
It includes four exercises covering webpage structure, forms, tables, and
HTML5 semantic elements.

## What’s Included
- Exercise 1: Basic webpage with text, lists, links, and images  
- Exercise 2: User form with inputs, radio buttons, checkboxes, and textarea  
- Exercise 3: Table with headings, rows, caption, and merged cells  
- Exercise 4: Semantic HTML using header, nav, article, aside, and footer  

## How to Run
1. Download or clone the repository
2. Open any `.html` file in a web browser
3. View the output
